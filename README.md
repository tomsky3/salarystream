## ✨ 项目简介

**薪流 (SalaryStream)** 是一个纯前端的实时薪资可视化与记账应用。输入你的月薪和工作时间，它会以**秒级精度**实时计算你正在赚取的金额，让你直观感受「时间的金钱价值」——然后帮你把这些收入管理起来：设定目标、记账消费、追踪流水。

**零依赖、零后端、开箱即用。** 所有数据存储在浏览器本地（localStorage），无需注册登录，打开即用。

## 🎯 功能特性

### 📊 主页 — 实时薪资仪表盘

| 功能 | 说明 |
|------|------|
| **⏱️ 秒级薪资计数器** | 每 100ms 刷新，精确到小数点后两位，看着余额跳动 |
| **🎨 10 种主题色** | 金色 / 翡翠 / 海洋蓝 / 玫瑰粉 / 紫罗兰 / 珊瑚橙 / 青碧 / 青柠 / 红宝石 / 浅色 |
| **🏆 目标管理** | 内置 20 种目标物品（房子/汽车/手机/旅行等），支持自定义添加、编辑、删除 |
| **📈 进度追踪** | 每个目标显示进度条 + 距离目标的剩余天数，达成时有彩色纸屑庆祝动画 |
| **💱 等价物翻译器** | 将当前收入翻译为具体物品：「你今天的工钱 = ☕ 3 杯星巴克」或「= 📱 0.03 个 iPhone」 |
| **⚙️ 灵活设置** | 自定义月薪、每月工作天数、每日工作时长 |
| **💾 数据安全** | 跨日自动存档、5 分钟检查点保存、关闭页面自动存档 |

### 💳 记账模块 — 消费商店 & 流水记录

| 功能 | 说明 |
|------|------|
| **🛒 一键消费商店** | 7 大消费分类（餐饮/交通/购物/娱乐/学习/日用/其他），点击即买 |
| **➕ 自定义记账** | 输入任意名称和金额，灵活记录每笔支出 |
| **💰 存款系统** | 「存入全部收入」（从实时计数器转入）或自定义金额存入 |
| **❄️ 飘字特效** | 每笔消费红色 `-¥xx` 从卡片飘起，存款绿色 `+¥xx` 上浮 |
| **📜 流水列表** | 最近 200 条交易记录，显示时间戳和操作类型 |
| **✏️ 编辑记录** | 修改任意交易的名称和金额，差额自动退款或补扣 |
| **🗑️ 删除退款** | 删除交易后对应金额自动返还到余额 |
| **⚠️ 余额不足提示** | 卡片抖动 + 黄色浮字提示差多少 |

### 🛠️ 技术细节

- **架构**: 单文件 HTML/CSS/JS（主页 `index.html` + 记账 `bookkeeping.html`）
- **字体**: [Outfit](https://fonts.google.com/specimen/Outfit)（展示）+ [Space Mono](https://fonts.google.com/specimen/Space+Mono)（数字等宽）
- **数据持久化**: `localStorage`（跨页面共享薪资配置）
- **动画**: CSS Keyframes + 原生 JS DOM 操作
- **响应式**: 适配桌面端和移动端（600px 断点）
- **无依赖**: 不使用任何前端框架或构建工具

## 🚀 使用方法

### 方式一：在线使用（推荐）

直接访问 GitHub Pages 部署地址：

**👉 https://tomsky3.github.io/salarystream/**

### 方式二：本地运行


## 🎮 操作指南

### 第一步：设置薪资

1. 打开主页，在顶部设置栏输入**月薪**
2. 设置**每月工作天数**和**每天工作时长**
3. 点击「开始」，实时计数器立即启动

### 第二步：设定目标

1. 在目标区域浏览内置的 20 个物品
2. 点击任意目标查看进度
3. 使用底部按钮**新增自定义目标**

### 第三步：开始记账

1. 点击主页金色「记账」按钮进入记账页面
2. 先「存入全部收入」将已赚金额转为可用余额
3. 在商店中点击商品即可消费，或使用「自定义记账」

### 第四步：管理流水

1. 悬停流水记录可看到**编辑**和**删除**按钮
2. 编辑可修改金额，差额自动调整余额
3. 删除会全额退款，带确认弹窗
